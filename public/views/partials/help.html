F.A.Q

Comment les points sont-ils distribués ?

Pour chaque pronostique, 3 cas de figures sont possibles :

  • Vous avez mis le score exacte (ex : 2-0 pronostiqué, 2-0 en réalité), alors dans ce cas vous receverez 40 points
  • Vous avez un écart identique (ex : 2-0 pronostiqué, 3-1 en réalité), alors dans ce cas vous receverez 20 points
  • Vous avez le bon vainqueur (ex : 2-0 pronostiqué, 1-0 en réalité), alors dans ce cas vous receverez 10 points
S'ajoute à cela, les paris bonus :
  • Pour une équipe en finale et à la bonne place : 100 points
  • Pour une équipe en finale mais pas à la bonne place : 50 points

Les scores seront-ils rafraichis en direct ?

Oui les scores seront rafraichis avec un interval d'une minute.

Est-il possible d'inviter mes amis sur Kupi.fr ?

Oui vous avez la possibilité de créer un salon puis de partager le lien d'inscription à ce salon.

J'ai trouvé un bug ou j'ai une idée d'amélioration, est-il possible de contacter l'équipe ?

Oui vous avez la possibilité d'ouvrir une issue sur le projet Github de Kupi. L'adresse du Github est la suivante : https://github.com/clemh78/worldcup
